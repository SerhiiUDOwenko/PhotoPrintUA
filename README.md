# 📸 PhotoPrint UA (v2.0)

**PhotoPrint UA** — це легкий та кросплатформений інструмент на базі Python для автоматизації підготовки фотографій до друку. Програма дозволяє швидко розкласти цифрові знімки на аркуші стандартних форматів (А4 або 10х15) з урахуванням офіційних вимог до документів та особових справ в Україні.

---

## ✨ Основні можливості

- **📏 Підтримка стандартів України:**
  - 3х4 см (Посвідчення, студентські квитки)
  - 3.5х4.5 см (Паспорт UA, закордонний паспорт, ID-картка)
  - 4х6 см (Військовий квиток, посвідчення офіцера)
  - 9х12 см (Особова справа)
  - 10х15 см (Альбомний формат)
- **📑 Гнучкість друку:** Вибір між папером **A4** та фотопапером **10x15 см (4x6")**.
- **🤖 Розумне заповнення (Fill Mode):** Якщо у папці лише одне фото, програма автоматично розрахує максимум і заповнить ним увесь аркуш.
- **🔢 Тиражування:** Можливість вказати точну кількість копій для кожної фотографії в папці.
- **✂️ Професійна сітка:** Суцільні вертикальні та горизонтальні лінії (2 px) для зручного розрізу лінійкою або роликовим різаком.
- **📝 Робота з текстом:**
  - Автоматичне перетворення назв файлів у **ВЕЛИКІ ЛІТЕРИ**.
  - **Auto-scaling:** Динамічне зменшення шрифту для довгих прізвищ, щоб текст ніколи не виходив за межі рамки.
- **🚀 Portable EXE:** Версія для Windows працює як один файл без необхідності встановлення Python та пошуку зовнішніх шрифтів.

---

## 🛠 Технічний стек

- **Мова:** Python 3.10+
- **Бібліотеки:** - `Pillow` (PIL) — обробка зображень, масштабування (crop/fit) та генерація PDF.
  - `Tkinter` — графічний інтерфейс користувача (GUI).
- **Якість виводу:** 300 DPI (стандарт високої чіткості для друку).

## 🚀 Як користуватися

### Для користувачів (Windows EXE):
1. Запустіть файл `PhotoPrint_UA.exe`.
2. У вікні налаштувань оберіть формат паперу, розмір фотографій та кількість копій.
3. Натисніть кнопку запуску та послідовно оберіть:
   - **Папку з оригіналами** (де лежать підготовлені фото).
   - **Папку для результату** (куди зберегти готовий PDF).
4. Відкрийте згенерований PDF та відправте на друк. 
   > **Важливо:** У налаштуваннях принтера обов'язково обирайте масштаб **"Реальний розмір" (Actual size / 100%)**, щоб зберегти точні міліметри!

### Для розробників (запуск із коду):
1. Клонуйте репозиторій.
2. Встановіть залежності:
   ```bash
   pip install Pillow
   ```
3. Запустіть скрипт:
   ```bash
   python photo_3x4_maker.py
   ```

## 📦 Збірка в EXE (для Windows)

Щоб створити автономний `.exe` файл (Portable) за допомогою PyInstaller, виконайте наступні кроки у терміналі Windows:

1. Встановіть необхідні пакети:
   ```powershell
   python -m pip install Pillow pyinstaller
   ```
2. Зберіть програму в один файл без консолі:
   ```powershell
   python -m PyInstaller --onefile --noconsole --name "PhotoPrint_UA" photo_3x4_maker.py
   ```

### 🎨 Додавання власної іконки

Щоб додати іконку (наприклад, згенерований логотип принтера), конвертуйте зображення у формат `.ico` (назвіть його `logo.ico`), покладіть у папку з проєктом і запустіть:
```powershell
python -m PyInstaller --onefile --noconsole --icon=logo.ico --name "PhotoPrint_UA" photo_3x4_maker.py
```
*Готовий файл з'явиться у папці `dist`.*
